文件权限：

-rw-r--r--

第一位表示文件类型： -（文件） d（目录） l（软链接文件）

第二到第四位表示所有者，第五到第七位表示所属组，第八到第十位表示其他用户

r:读 w:写 x:执行

安装mysql

1.linux默认有mariadb,先执行 yum install mysql

2.删掉mysql yum remove mysql 检查 rpm -qa | grep mysql 还有就删掉

3.下载mysql包wget http://repo.mysql.com/mysql57-community-release-el7-8.noarch.rpm

4.更新安装rpm -Uvh mysql57-community-release-el7-8.noarch.rpm

5.安装mysql-server yum install mysql-server

6.第一次安装，改mysql密码 重启service mysqld restart

7.登录mysql -u root 改密码 update user set password=password("新密码") where user="root";

8.改完密码后 service mysqld restart

9.登录mysql -u root -p 输入新密码
